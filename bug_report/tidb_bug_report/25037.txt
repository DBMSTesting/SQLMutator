## Bug Report

### 1. Minimal reproduce step (Required)
```
set global tidb_enable_change_column_type = 1;

drop table tbl_3;
create table tbl_3 ( col_19 bit(13) not null) collate utf8mb4_general_ci;
insert into tbl_3 values ( 4047 );
alter table tbl_3 change column col_19 col_73 binary(248) collate binary default 't';

show create table tbl_3;
select * from tbl_3;
```
