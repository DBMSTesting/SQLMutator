Description:
## Bug Report

Please answer these questions before submitting your issue. Thanks!

1. What did you do?
If possible, provide a recipe for reproducing the error.
```
 SELECT -1 % 0.5;
 SELECT -0.0 = (-1 % 0.5);
```

2. What did you expect to see?
In MySQL 8.0.15
```
mysql> SELECT -1 % 0.5;
+----------+
| -1 % 0.5 |
+----------+
|      0.0 |
+----------+
1 row in set (0.00 sec)

mysql> 
mysql> SELECT -0.0 = (-1 % 0.5);

| -0.0 = (-1 % 0.5) |
+-------------------+
|                 1 |
+-------------------+
1 row in set (0.00 sec)
```

```
mysql> SELECT -1 % 0.5;

| -1 % 0.5 |

|     -0.0 |

mysql>  SELECT -0.0 = (-1 % 0.5);

| -0.0 = (-1 % 0.5) |
+-------------------+
|                 0 |
+-------------------+
1 row in set (0.00 sec)

```
