3. Import TPCH 50g data
4. Analyze all tables manually
5. check the row_count of lineitem table and explain analyze Query 18 before the auto analyzing is finished
6. check the row_count of lineitem table and explain analyze Query 18 after the auto analyzing is finished
```
/*PLACEHOLDER*/ select
	c_name,
	c_custkey,
	o_orderkey,
	o_orderdate,
	o_totalprice,
	sum(l_quantity)
from
	customer,
	orders,
	lineitem
where
	o_orderkey in (
		select
			l_orderkey
		from
			lineitem
		group by
			l_orderkey having
				sum(l_quantity) > 314
	)
	and c_custkey = o_custkey
	and o_orderkey = l_orderkey
group by
	c_name,
	c_custkey,
	o_orderkey,
	o_orderdate,
	o_totalprice
order by
	o_totalprice desc,
	o_orderdate
limit 100;
```
