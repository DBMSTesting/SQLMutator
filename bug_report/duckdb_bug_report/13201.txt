Title: [Explain] Add the `GRAPHVIZ` format for `EXPLAIN` statements
Description:
`EXPLAIN (FORMAT GRAPHVIZ) SELECT ...`

Will output [graphviz](https://graphviz.org/) data that can be rendered by external tools.
